# Demo Video Script

## Flowbit Analytics Dashboard - Demo Video

### Duration: 3-5 minutes

### Sections:

1. **Introduction (30 seconds)**
   - Show the dashboard homepage
   - Explain what the application does
   - Highlight key features: Analytics Dashboard + Chat with Data

2. **Dashboard Overview (1 minute)**
   - Show the 4 overview cards (Total Spend YTD, Invoices Processed, Documents Uploaded, Avg Invoice Value)
   - Explain how data is fetched from the backend API
   - Show the charts:
     - Invoice Volume + Value Trend (line chart)
     - Spend by Vendor (Top 10 horizontal bar)
     - Spend by Category (pie/donut chart)
     - Cash Outflow Forecast (bar chart)

3. **Invoices Table (30 seconds)**
   - Show the searchable, sortable invoices table
   - Demonstrate search functionality
   - Show sorting by date and amount
   - Show pagination

4. **Chat with Data Feature (2 minutes)**
   - Switch to the "Chat with Data" tab
   - Ask a question: "What's the total spend in the last 90 days?"
   - Show the generated SQL query
   - Show the results table
   - Ask another question: "Show me the top 5 vendors by spend"
   - Show the SQL and results
   - Demonstrate the explanation feature

5. **Architecture Overview (30 seconds)**
   - Show the architecture diagram from README
   - Explain the flow: Frontend → API → Vanna AI → Database
   - Highlight security features (read-only DB user, SQL sanitization)

6. **Conclusion (30 seconds)**
   - Summarize key features
   - Mention deployment URLs
   - Thank viewers
